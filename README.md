# Reservation System (Full‑Stack Web Application)

## 📌 Project Description
A full‑stack reservation system where users can:
- Register and log in securely
- Search available routes
- Book and cancel reservations
- Manage their profile and view booking history
- Access admin dashboard for route and user management

This project demonstrates end‑to‑end full‑stack development with modern UI, secure backend APIs, and professional deployment practices.

---

## 🛠 Tech Stack
**Frontend:**
- React + TypeScript
- Material‑UI (MUI) for modern UI components
- React Router for navigation
- Axios for API calls

**Backend:**
- Spring Boot (REST APIs)
- JWT Authentication & Role‑based access
- JPA/Hibernate for persistence
- MySQL database

**DevOps & Extras:**
- Docker for containerized deployment
- GitHub Actions for CI/CD pipeline
- Postman for API testing
